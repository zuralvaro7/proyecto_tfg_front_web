# Aplicación Web

La aplicación web es accesible desde cualquier dispositivo con conexión a Internet a través de un navegador web. Para acceder, introduce la URL de la aplicación en la barra de direcciones de tu navegador.

---

## Registro de usuario

Si es la primera vez que utilizas la aplicación, deberás crear una cuenta siguiendo estos pasos:

1. Accede a la URL de la aplicación.
2. Haz clic en **¿No tienes una cuenta? Regístrate aquí**.
3. Completa el formulario de registro con los datos solicitados.
4. Pulsa el botón **Enviar**.
5. Revisa tu correo electrónico y abre el mensaje de verificación recibido.
6. Accede al enlace de verificación incluido en el correo.
7. Una vez verificada la cuenta, recarga la página para poder iniciar sesión.

---

## Inicio de sesión

Para acceder a tu cuenta:

1. Introduce tu nombre de usuario.
2. Introduce tu contraseña.
3. Pulsa el botón **Enviar**.

Si las credenciales son correctas, serás redirigido automáticamente a la pantalla principal de historial.

---

## Historial de visualización

Una vez iniciada la sesión, podrás acceder al historial de episodios visualizados desde la pantalla principal de la aplicación.

En esta sección se muestran los registros almacenados por la aplicación de escritorio, permitiendo consultar el seguimiento realizado de las series.

---

## Edición del perfil

La aplicación permite modificar determinados datos de la cuenta de usuario.

### Datos modificables

* Nombre de usuario.
* Dirección de correo electrónico.

### Opciones adicionales

Desde la pantalla de perfil también es posible:

* Cerrar la sesión actual.
* Eliminar la cuenta de usuario de forma permanente.

> **Importante:** La eliminación de la cuenta es irreversible y supondrá la pérdida de todos los datos asociados al usuario.
